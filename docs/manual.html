Codex Conductor

A local cockpit, CLI, and MCP server for watching Codex sessions from ~/.codex/sessions.

Commands

codex-conductor
codex-conductor --minutes 120
codex-conductor --all
codex-conductor --json
codex-conductor up
codex-conductor mcp

Managed Windows

codex-conductor run review
codex-conductor say review "continue"
codex-conductor attach review
codex-conductor stop review

Managed windows run in tmux under the codex-conductor session. Plain sessions are read-only until adopted into tmux.

Labels

Customize project labels in ~/.codex-conductor/labels.json.

MCP

codex-conductor mcp exposes tools for listing sessions, summarizing one session, checking inferred remaining work, finding pending questions, and driving managed windows through the irreversibility gate.

Safety

The cockpit binds to localhost. State-changing HTTP calls require a local origin and X-Conductor: 1. auto_continue refuses unattended deploy, send, delete, and spend approvals.
